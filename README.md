#  Sistema de Gestão de Inventário - TI
Sistema desenvolvido em **Python** com banco de dados **SQLite** para controle de estoque de hardware.

## O que este projeto faz:
*   **CRUD Completo:** Permite cadastrar, consultar, atualizar e excluir itens do estoque.
*   **Inteligência de Dados:** Soma quantidades automaticamente se o item já existir e alerta quando uma peça está esgotada.
*   **Segurança e Padronização:** Uso de `.strip()` e `.upper()` para evitar duplicidade e garantir a integridade do banco.

### Tecnologias Utilizadas:
*   Python 3
*   SQLite3 (Banco de Dados Relacional)
